Aqui será descrito os padrões usados no código Javascript da interface Web.

Padrões:

  • Editor padrão: VI ou VIM;
  • TUDO em inglês,
  • Todos os comandos terminam ";",

  • Função:
    • Nome da função no formato: MODULE_FunctionName(),
    • Parametros de função com a primeira letra em maíusculo: FunctionName(Pname1, Pname2),
    • Abrir com "{" na linha seguinte a declaração da função e fechar com "}" uma linha após o último comando,
    • Variáveis locais com a primeira letra de cada palavra maiúsculo,
    • Return sem parenteses: return value,
    • Declarar as váriáveis por primeiro, pode ocorrer atribuições simples,

  • Comentários:
    • Colocar em todos os arquivos o cabeçalho GPL,
    • Colocar uma descrição sobre o arquivo após o cabeçalho GPL,
    • Toda função tem que ter uma descrição, o valor retornado e se é público ou privado (ver exemplo),
    • Colocar comentário sobre as características das funçoes, exemplo funções que manipulam dados na interface.

  • If, while:
    • As condições tem que estar entre parenteses. Se houver mais de uma condição, colocar cada uma entre parenteses.
    • Sempre coloca chaves para indicar os comandos. Abrir com "{" após a condição do if/while e fechar com "}" após o último comando, mantendo a edentação(?).

  • Switch/Case: -
